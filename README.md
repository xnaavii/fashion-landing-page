# Fashion Landing Page

A modern, responsive fashion landing page built with HTML, CSS, and Vanilla JavaScript.  
The project was created to showcase design sensibility, UI layout skills, and frontend interaction, rather than backend functionality.

[Check the deployed site](https://xnaavii.github.io/fashion-landing-page/)

## Purpose

This project focuses on:

- Visual presentation
- Layout and spacing
- Smooth UI interactions
- Clean, readable frontend code

It serves as a design-forward portfolio piece demonstrating how a simple tech stack can still produce a polished, engaging user experience.

## Tech Stack

- HTML5 – semantic markup
- CSS3 – layout, styling, responsiveness
- JavaScript (ES6) – DOM manipulation and UI logic

No frameworks or libraries were used.

## Features

- Scrolling product slider
- Interactive navigation dots
- Product Details
- Newsletter subscription section
- Responsive layout

## Design Focus

- Clear visual hierarchy
- Consistent spacing and typography
- Emphasis on product imagery
- Minimal, modern UI
- Smooth transitions and interactions

The goal was to create a clean fashion-focused aesthetic that could realistically be used as a brand landing page.

## Colors and typography

- [Open color link](https://yeun.github.io/open-color/#gray)
- Inter was used as a base and Helvetica for headings
